# BundleAI

**Turn any sales CSV into money-making product bundles — in 30 seconds**

Fully automatic • No code needed • Powered by FP-Growth (the same algorithm Amazon & Walmart use)

Live app: https://bundleai-freda.streamlit.app

### Features
- Upload any transaction CSV (comma-separated baskets)
- Instantly get: Top items • Product Love Map • Elite bundles ranked by revenue impact
- Hidden gems & customer segment detection
- One-click CSV download

pip install -r requirements.txt
